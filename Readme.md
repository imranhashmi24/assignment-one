# DevConf 2026

A modern and responsive landing page for **DevConf 2026**, built using **HTML5** and **CSS3**. The project showcases a clean conference website with sections for speakers, pricing, career opportunities, and footer.

## Live Website

https://your-live-link.netlify.app/

## Features

- Responsive Navigation Bar
- Hero Banner Section
- Meet the Speakers Section
- Pricing Plans Section
- Developer Career Hub (Unique Section)
- Responsive Footer
- Mobile-Friendly Layout
- Clean and Modern UI

## Technologies Used

- HTML5
- CSS3
- Font Awesome
- Google Fonts (Arimo)
